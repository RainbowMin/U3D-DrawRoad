2D Toolkit Documentation

Welcome to the 2D Toolkit documentation.

2D Toolkit is a set of tools to provide an efficient 2D sprite, collider set-up and text system which integrates seamlessly into the Unity environment. This manual contains a quick-start walkthrough, and a reference for the various interfaces in the package.

This documentation was generated from 2D Toolkit 2.10 final + hotfix 1. Click here to download an offline copy of the documentation.

  • 2D Tookit UI

    The 2D Toolkit UI system is a fast, efficient and easy to use UI system built speficially for 2D Toolkit. It integrates seamlessly into the rest of the system - allowing you to create UI for your games quickly, sharing sprites and atlases with the rest of your game without incurring any additional overheads.
